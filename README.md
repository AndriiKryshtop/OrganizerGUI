# OrganizerGUI

Organizer with GUI (Java FX), Maven and MVC.
Application that helps you to organize your time and dont forget important dates.
It have notification system.

# Created on:
Java 1.8;
Maven;
GUI: JavaFX;
Log4j + slf4j.

# Used patterns: MVC, Observer, Singleton and few more.
